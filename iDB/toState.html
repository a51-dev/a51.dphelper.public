toState

Save all DB and Tables to Global State

  • You can use (as example): 'dphelper.iDB.toState( 'test', dphelper.iDB.get( 'test', 'test', 1 ) )'
  • or: 'dphelper.iDB.toState()' to load all indexedDB into dphelper.databases global store.
Please type 'dphelper.databases' in console to see all DBs
